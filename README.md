# extra VTi repository (mips32el)
Some extra/replacment packages for VTi

## Installation
- Login on VTi Box
- `cd /etc/opkg`
- `wget https://raw.githubusercontent.com/cfdisk/vti-banana-mipsel/master/vti-banana-mipsel-feed.conf`
- `opkg update`
